# Student-Management-System
This Student Management System is a Java-based console application that manages student records using JDBC and MySQL. It supports adding, viewing, and deleting student details while demonstrating Core Java, OOP concepts, prepared statements, and database connectivity.
# Student Management System

A Java-based console application to manage student records using JDBC and MySQL.

## Features
- Add new students
- View all students
- Delete students

## Technologies Used
- Java
- JDBC
- MySQL

## How to Run
1. Create database and table using provided SQL.
2. Update DB credentials in DBConnection.java.
3. Compile and run Main.java.
